Top level: classes, interfaces, enums

* public:
    Top level: the object is visible to all classes everywhere, whether they are in the same package or have imported the package containing the public class.
    Member level: at the member level, public has the same meaning as at top level. A public class member (or filed) and public method can be accessed from any other class anywhere, even in different package.

* package private (default -> no keyword)
    Top level: the object is only available within its own package (and is visible to every class within the same package).
    Member level: this also the same meaning as it does at top level. An object with no access modifier is visible to every class within the same package (but not to classes in external packages).

Only member level:
* protected: the object is visible anywhere in its own package (like package-private) but also in subclasses even if they are in another package.
* private: the object is only visible within the class it is declared. It is not visible anywhere else (including in subclasses of its class).
